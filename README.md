## 下载并运行官方文档

- 进入GitHub：<https://github.com/mrdoob/three.js/>
  - 切换至分支`dev`
  - 选择右侧`code`
  - 选择`download ZIP`
- 解压项目并进入目录内
- `npm install`进行安装(首次)
- `npm run start`启动项目
- 访问地址：<http://localhost:8080/>
  - `docs`文档
  - `editor`可视化编辑器
  - `examples`示例
  - `manual`手册
